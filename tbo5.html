Regular Expressions: Using the Test Method

Regular Expressions digunakan dalam bahasa pemrograman untuk mencocokkan bagian dari string. Kamu dapat membuat suatu pola untuk membantu kamu melakukan pencocokan dalam string.

Contoh :

Jika Anda ingin menemukan kata "Teknik" dalam string "DepartemenTeknikInformatika", Kamu dapat menggunakan Regular Expressions berikut: / Teknik /. Perhatikan bahwa tanda kutip tidak diperlukan dalam Regular Expressions.

JavaScript memiliki banyak cara untuk menggunakan Regular Expressions. Salah satu cara untuk menguji regex adalah menggunakan method .test(). Method .test() mengambil regex dan menerapkannya ke string (yang ditempatkan di dalam tanda kurung), dan akan mengembalikan sebuah nilai true atau false jika pola yang kamu temukan benar atau tidak.
